# Seedance MCP — AI 视频生成

> 让 Claude Code 直接调用火山引擎 Seedance API 生成 AI 视频。

## 定价策略

| 方案 | 价格 | 包含 |
|------|------|------|
| Free | $0 | 每月 5 条 720p 视频 |
| Pro | $15/月 | 每月 50 条 1080p + 图生视频 |
| Pay-as-you-go | $0.05/条 | 按量付费, 无上限 |

## 目标用户

- 自媒体创作者（抖音/小红书/TikTok 视频量产）
- AI 创业者（快速原型视频素材）
- 电商卖家（产品展示视频批量生成）

## 竞品对比

|  | Seedance MCP | Runway | 可灵 |
|------|------|------|------|
| 集成方式 | Claude Code 内对话 | Web UI | App/API |
| 价格 | $0.05/条起 | $12/月起 | $0.20/条 |
| 多模态输入 | 文本+图片+视频+音频 | 文本+图片 | 文本+图片 |
| 中文 prompt | 原生支持 | 翻译后 | 原生 |

## 技术亮点

- 基于火山引擎 Seedance 2.0（字节跳动）
- 支持四模态输入：文本 + 参考图 + 参考视频 + 参考音频
- 5-15 秒可调，720p/1080p，16:9 / 9:16 / 1:1
- MCP 标准协议，Claude Code / Codex CLI / Cursor 通用
